## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Sections

- **Hero**: Introduction and call-to-action
- **Experience**: Professional work history with detailed descriptions
- **Skills**: Technical skills with visual progress indicators
- **Education**: Academic background
- **Contact**: LinkedIn profile and contact information

## Customization

All content is stored in the component files. You can easily update:
- Personal information in `src/components/Hero.jsx`
- Work experience in `src/components/Experience.jsx`
- Skills in `src/components/Skills.jsx`
- Education in `src/components/Education.jsx`
- Contact information in `src/components/Contact.jsx`
